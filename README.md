# KIDP2025 밋업 플랫폼

2025 디자인-기술 협업 전주기지원 밋업 플랫폼입니다.

## 프로젝트 개요

기술기업과 디자인전문기업 간의 매칭 및 밋업 관리를 위한 웹 플랫폼입니다.

## 주요 기능

- 참가자 관리 및 CSV 가져오기
- 기업 간 선호도 설정 및 매칭
- 라운드별 테이블 배정
- 좌석 배치도 시각화
- 관리자 대시보드

## 기술 스택

- **프론트엔드**: HTML, CSS, JavaScript
- **백엔드**: Cloudflare Pages Functions (Hono)
- **데이터베이스**: Cloudflare D1
- **인증**: Google OAuth
